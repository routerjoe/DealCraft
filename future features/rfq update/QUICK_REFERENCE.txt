┌─ RFI/MRR RESPONSE STRATEGY ────────────────────────────────────────────────┐
│ ALWAYS respond:    Strategic customers (CRITICAL/HIGH)                    │
│ USUALLY respond:   $50K+ estimated future value                           │
│ EVALUATE:          $25K-$50K opportunities (case-by-case)                 │
│ USUALLY decline:   <$25K + non-strategic + high competition               │
└────────────────────────────────────────────────────────────────────────────┘

┌─ SCORING SYSTEM (0-100 points) ────────────────────────────────────────────┐
│ • Value (0-40):      $1M+=40, $200K+=30, $20K+=20, <$20K=10              │
│ • Customer (0-25):   CRITICAL=25, HIGH=15, Standard=0                     │
│ • Competition (0-15): <20=15, <50=10, <100=5, 100+=0                     │
│ • Technology (0-10):  HIGH priority=10, MEDIUM=5, Other=0                │
│ • OEM (0-10):        Authorized=10, Not authorized=0                      │
│ • Renewal (0-10):    Existing customer=10, New=0                          │
│                                                                            │
│ 75-100: GO - High Priority          45-59: REVIEW - Conditional GO        │
│ 60-74:  GO - Consider Pursuit       30-44: REVIEW - Likely NO-GO          │
│ 0-29:   NO-GO - Auto-Decline                                              │
└────────────────────────────────────────────────────────────────────────────┘

┌─ DECISION WORKFLOW ────────────────────────────────────────────────────────┐

│  New RFQ Email Received                                                   │

│  Check Strategic Flags    → YES → High priority review                    │
│         ↓ NO                                                               │
│  Calculate Score (0-100)                                                   │
│         ↓                                                                  │
│  75+:  GO - Immediate pursuit                                             │

│  45+:  REVIEW - Needs evaluation                                          │
│  30+:  REVIEW - Weak case                                                 │
│  <30:  NO-GO - Decline                                                    │
│                                                                            │
│  Log Decision + Track OEMs                                                │
│                                                                            │
└────────────────────────────────────────────────────────────────────────────┘
